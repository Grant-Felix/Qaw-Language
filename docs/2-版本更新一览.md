# 版本更新一览

> 记录妖文编程语言每次更新的**新增 / 弃用 / 移除**项。
> 与 [`1-妖文编程语言从入门到精通`](1-妖文编程语言从入门到精通.md) 配套使用。

## 使用方法

每次对蓝图或实现做更新时，新增一行到下方表格：

| 版本 | 日期 | 类型 | 描述 | 影响范围 |
|:---|:---|:---|:---|:---|
| 0.0.0 | YYYY-MM-DD | 新增/弃用/移除 | 一句话描述 | 章节 / 文件 / API |
