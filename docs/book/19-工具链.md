## 19.2 `qawc` 命令

```bash
qawc version                        # 打印版本
qawc lex main.qaw                   # 词法分析
qawc parse main.qaw                 # 语法分析 → AST
qawc run main.qaw                   # 解释执行（v0.1 默认）
qawc build main.qaw -o out          # 编译为可执行（v0.5+）
qawc check main.qaw                 # 仅类型检查（v0.5+）
qawc expand main.qaw                # 宏展开（v0.9+）
```

## 19.3 `qawpm` 命令

```bash
qawpm init my_project
qawpm add yao.http
qawpm add yao.contrib.serde --features=json
qawpm remove yao.http
qawpm update --dry-run
qawpm install
qawpm build --release
qawpm test
qawpm publish
```

## 19.4 `qaw.mod`

```toml
[package]
name = "my_project"
version = "1.0.0"
edition = "2026"
authors = ["张三 <zhangsan@example.com>"]
license = "Apache-2.0"

[lib]
name = "my_project"
path = "src/lib.yao"

[[bin]]
name = "my_cli"
path = "src/main.yao"

[dependencies]
"yao.std.http" = "1.0"
"yao.contrib.serde" = { version = "1.5", features = ["json"] }

[features]
default = ["std"]
async = ["yao.contrib.tokio"]
serde = ["yao.contrib.serde"]

[profile.dev]
opt_level = 0
debug = true

[profile.release]
opt_level = 3
lto = "thin"
```

## 19.5 `qaw fmt`

```bash
qaw fmt                   # 格式化当前目录
qaw fmt --check src/      # 检查但不修改
qaw fmt --diff src/       # 显示 diff
```

配置文件 `.yaofmt`：

```toml
[fmt]
max_width = 100
tab_spaces = 4
trailing_comma = "vertical"
```

## 19.6 `qaw doc`

```bash
qaw doc                        # 生成 HTML
qaw doc --open                 # 浏览器打开
qaw test --doc                 # 运行文档测试
```

## 19.7 `qaw clippy`

```bash
qaw clippy
qaw clippy --fix               # 自动修复
```

## 19.8 `qaw test`

```bash
qaw test                       # 运行所有测试
qaw test --doc                 # 仅文档测试
qaw test --test-threads=4      # 并行
```

```yao
#[test]
fn test_basic() {
    assert_eq!(add(1, 2), 3);
}

#[test]
#[should_panic]
fn test_div_by_zero() {
    divide(1, 0);
}

#[test_case(1, 2, 3)]
#[test_case(0, 0, 0)]
fn test_add(a: int, b: int, expected: int) {
    assert_eq!(add(a, b), expected);
}
```

## 19.9 `yao lsp`

```bash
yao lsp                    # stdio 模式
yao lsp --port 9999        # TCP 模式
```

支持的 LSP 功能：补全、跳转定义、查找引用、重命名、格式化、错误检查、悬停文档、签名帮助、重构。

## 19.10 调试器

```bash
qaw debug program
qaw debug --attach <pid>
```

gdb 风格命令：

```
break main
break src/main.yao:42
run
next
step
continue
print x
backtrace
watch x
finish
quit
```
