## 18.1 总体架构

```
┌─────────────────────────────────────────────┐
│              应用代码                         │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐        │
│  │ 协程 G1 │ │ 协程 G2 │ │ 协程 G3 │ ...    │
│  └─────────┘ └─────────┘ └─────────┘        │
├─────────────────────────────────────────────┤
│  调度器 (Scheduler / GMP)                    │
├─────────────────────────────────────────────┤
│  异步执行器 (Executor / Future 驱动)          │
├─────────────────────────────────────────────┤
│  反应器 (Reactor / epoll/kqueue/IOCP)         │
├─────────────────────────────────────────────┤
│  内存管理（@safe 默认无 GC，@dynamic 有 GC）   │
└─────────────────────────────────────────────┘
              ↓
        OS / 硬件
```

## 18.2 GMP 调度器

- **G（Goroutine）**：用户级协程。
- **M（Machine）**：OS 线程。
- **P（Processor）**：逻辑处理器。
- 工作窃取调度。

```bash
yao run --runtime.gomaxprocs=8 main.yao
```

## 18.3 协程栈

- 初始大小 **8KB**。
- 按需增长（拷贝到 2 倍大小）。

## 18.4 异步运行时

- 反应器：epoll（Linux）/ kqueue（macOS）/ IOCP（Windows）。
- 阻塞 IO → spawn_blocking 转异步。
- Future 与 Task 调度。

## 18.5 通道

- MPMC（多生产者多消费者）。
- 缓冲 / 无缓冲。
- 关闭后接收剩余元素。
- 原子操作。

## 18.6 GC（仅 @dynamic）

| GC | 适用 |
|:---|:---|
| serial | 单线程 |
| concurrent | 多线程并发标记 |
| generational | 长寿对象优化 |

```bash
qawc build --gc=concurrent
```

## 18.7 WASM 运行时

- 单线程（默认）。
- 异步基于 JS event loop。
- 通过 SharedArrayBuffer 支持有限的多线程。
