## 3.1 安装

蓝图阶段（v0.0）：无需安装，仅设计文档。

v0.1+ 阶段：

```bash
# macOS / Linux
curl -sSf https://yao-lang.org/install.sh | sh

# Windows
iwr https://yao-lang.org/install.ps1 | iex

# 包管理器
brew install yao
```

## 3.2 第一个程序：Hello 妖文

文件 `hello.yao`：

```yao
package hello;

func main() -> void {
    print("你好, 世界!");
}
```

运行：

```bash
$ yaoc run hello.yao
你好, 世界!
```

## 3.3 四形制关键字演示

```yao
package hello;

// 四种书写方式完全等价
func greet() -> void { print("hi"); }
fn greet2() -> void { print("hi"); }     // 缩写
hanshu greet3() -> void { print("hi"); } // 全拼
hs greet4() -> void { print("hi"); }     // 首字母

func main() -> void {
    greet();
    greet2();
    greet3();
    greet4();
}
```
