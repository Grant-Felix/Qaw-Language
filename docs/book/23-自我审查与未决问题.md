## 23.1 设计冲突（待 RFC 解决）

| 编号 | 议题 | 倾向 |
|:---|:---|:---|
| ⚠️ RFC-0001 | 关键字首字母冲突（如 `fh`、`sx`） | 已倾向解决，待锁定 |
| ⚠️ RFC-0002 | `goto` 是否保留 | 保留但限制使用 |
| ⚠️ RFC-0003 | `!` 多义（宏尾 / 强制解包 / 逻辑非） | 上下文消歧 |
| ⚠️ RFC-0004 | `importPython` 命名 | `importPython` 关键字 |
| ⚠️ RFC-0005 | `@unsafe` vs `unsafe` | 仅 `unsafe` |
| ⚠️ RFC-0006 | `let` vs `var` | `let` 不可变，`let mut` / `var` 可变 |
| ⚠️ RFC-0007 | trait 演进策略 | 默认方法 + 新 trait |
| ⚠️ RFC-0008 | 协程调度器细节 | GMP |
| ⚠️ RFC-0009 | GC 选型 | 默认 concurrent |
| ⚠️ RFC-0010 | 自举路径细节 | C → Yao → Yao |
