## B.1 优先级（从高到低）

| 优先级 | 运算符 | 描述 | 结合性 |
|:---:|:---|:---|:---|
| 1 | `=` `+=` `-=` `*=` `/=` `%=` `&=` `\|=` `^=` `<<=` `>>=` | 赋值 | 右 |
| 2 | `\|\|` | 逻辑或 | 左 |
| 3 | `&&` | 逻辑与 | 左 |
| 4 | `==` `!=` `<` `<=` `>` `>=` | 比较 | 不结合 |
| 5 | `\|` | 按位或 | 左 |
| 6 | `^` | 按位异或 | 左 |
| 7 | `&` | 按位与 | 左 |
| 8 | `<<` `>>` | 移位 | 左 |
| 9 | `+` `-` | 加减 | 左 |
| 10 | `*` `/` `%` | 乘除模 | 左 |
| 11 | `..` `..=` | 范围 | 不结合 |
| 12 | `as` | 类型断言 | 左 |
| 13 | `?` `?.` `??` | 错误传播/可选链/空值合并 | 右 |
| 14 | `\|args\|` | 闭包 | 右 |
| 15 | `-` `!` `*` `&` `&mut` `not` | 一元 | 右 |
| 16 | `()` `[]` `.` `::` `?` `!` | 后缀 | 左 |

## B.2 算术

```yao
let a = 10 + 5;      // 15
let b = a * 2;       // 30
let c = a / 3;       // 5（整数截断）
let d = a % 3;       // 1
let e = -a;          // -15
```

字符串拼接：`"hello" + ", " + "world"`。

## B.3 比较

```yao
let f = a == b;      // false
let g = a < b;       // true
let h = !g;          // false
```

**不**支持链式比较（与 Python 不同）：`0 < x < 10` ❌，应写 `0 < x && x < 10`。

## B.4 逻辑

```yao
let p = !cond;       // 逻辑非
let q = a > 0 && b > 0;     // 逻辑与（短路）
let r = a > 0 || b > 0;     // 逻辑或（短路）
```

## B.5 按位

```yao
let x = 0xFF & 0x0F;     // 0x0F
let y = 0x01 | 0x02;     // 0x03
let z = 0x01 << 4;       // 0x10
```

## B.6 范围

```yao
for i in 0..10 { ... }      // 0, 1, ..., 9（半开）
for i in 0..=10 { ... }     // 0, 1, ..., 10（闭）
```

## B.7 `as` 类型断言

```yao
let a = 100i32;
let b = a as i64;        // widening
let c = a as i8;         // narrowing（截断）
```

## B.8 `?` `?.` `??` `!`

```yao
let v = may_fail()?;            // 错误传播
let c = user?.address?.city;    // 可选链
let n = opt ?? "default";       // 空值合并
let x = some_option!;           // 强制解包（None 时 panic）
```

## B.9 通道

```yao
c <- 42;                // 发送
let v = <-c;             // 接收
```
