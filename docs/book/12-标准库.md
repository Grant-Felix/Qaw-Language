## 12.1 模块总表

```
yao.std
├── prelude     # 默认导入
├── core        # 核心 trait（no_std 兼容）
├── alloc       # 堆分配
├── collections # 集合
├── string      # 字符串
├── io          # IO
├── fs          # 文件系统
├── net         # 网络
├── http        # HTTP
├── sync        # 同步原语
├── chan        # 通道
├── thread      # 线程
├── async       # 异步运行时
├── context     # 上下文
├── runtime     # 协程调度
├── time        # 时间
├── path        # 路径
├── num         # 数值扩展
├── math        # 数学
├── ptr         # 指针
├── slice       # 切片
├── iter        # 迭代器
├── unicode     # Unicode
├── fmt         # 格式化
├── log         # 日志
└── proc_macro  # 过程宏 API
```

## 12.2 prelude

自动导入常用项：

```yao
// 自动 use，无需手动
Option, Some, None,
Result, Ok, Err,
Display, Debug, Clone, Copy,
Eq, PartialEq, Ord, PartialOrd,
Hash, Default, Iterator, IntoIterator,
From, Into, AsRef, AsMut,
Send, Sync,
Add, Sub, Mul, Div, Rem,
Drop, size_of, panic!, etc.
```

## 12.3 核心 API 示例

```yao
// Vec
let v = vec![1, 2, 3];
v.push(4);
let first = v.first().unwrap();

// HashMap
let mut m = HashMap::new();
m.insert("key", "value");
let v = m.get("key");

// String
let s = "hello".to_uppercase();
let s = format!("name: {}, age: {}", name, age);
```
