## 5.1 if 表达式

```yao
let sign = if x > 0 { "正" }
          else if x < 0 { "负" }
          else { "零" };
```

`if` 是**表达式**，有类型有值。

## 5.2 循环

### 计数循环糖

```yao
for i to 10 { print(i); }           // 0..10
for i from 1 to 10 { print(i); }    // 1..10
for i from 10 downto 1 { print(i); } // 10..=1 反向
for i from 0 to 100 step 2 { ... }  // 0, 2, 4, ...
```

### 迭代循环

```yao
for item in arr { print(item); }
for (i, item) in arr.enumerate() { print("${i}: ${item}"); }
```

### while 循环糖

```yao
while cond {
    // ...
}
```

### 无限循环

```yao
loop {
    if done { break; }
}
```

## 5.3 模式匹配

```yao
match value {
    0 => print("零"),
    1 | 2 | 3 => print("小"),
    n if n < 0 => print("负"),
    _ => print("其它"),
}
```

- 穷尽所有分支。
- `_` 是通配符。

## 5.4 跳转

```yao
break;            // 跳出循环
continue;         // 跳过本轮
break 'outer;     // 跳出标签循环
return value;     // 返回值
```

## 5.5 defer

```yao
func process() {
    let f = open("file.txt");
    defer f.close();    // 函数结束时调用
    
    // ...
}
```

`defer` 按 LIFO 顺序执行。
