## 16.2 文档注释

```yao
/// 计算两个数的最大公约数。
///
/// # 示例
///
/// ```
/// let g = gcd(12, 8);
/// assert(g == 4);
/// ```
///
/// # 复杂度
///
/// O(log(min(a, b))).
func gcd(a: int, b: int) -> int {
    if b == 0 { a } else { gcd(b, a % b) }
}
```
