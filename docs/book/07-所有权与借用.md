## 7.1 所有权三大铁律

1. 每个值有且仅有一个所有者。
2. 当所有者离开作用域，值被立即 drop。
3. 借用规则：
   - 共享借用 `&T` 可多个
   - 可变借用 `&mut T` 最多一个
   - 共享与可变借用不能共存

## 7.2 移动语义

```yao
let s1 = String::from("hello");
let s2 = s1;       // s1 移动到 s2
// print(s1);      // ❌ 编译错误：s1 已无效
```

## 7.3 Copy 类型

```yao
let x = 5;
let y = x;          // i32 是 Copy，按位复制
print(x);            // ✅ x 仍可用
```

内置 `Copy`：`i8`-`i128`, `u8`-`u128`, `isize/usize`, `f32/f64`, `bool`, `char`, `byte`, `&T`。

## 7.4 借用

```yao
let x = 5;
let r: &int = &x;        // 共享借用
print(*r);                // 5（通过解引用读）

let mut y = 10;
let mr: &mut int = &mut y;
*mr = 20;                 // 通过解引用写
```

## 7.5 智能指针

```yao
let b = Box::new(5);           // 堆分配
let a = Arc::new(vec![1,2]);  // 多线程原子引用计数
let r = Rc::new(vec![1,2]);   // 单线程引用计数
```

## 7.6 内部可变性

```yao
struct Cell {
    value: Ref<int>,
}

impl Cell {
    method set(mut self, v: int) {
        *self.value.borrow_mut() = v;
    }
    
    method get(self) -> int {
        *self.value.borrow()
    }
}
```
