## 11.1 包

一个包 = 一个 `yao.mod` + 多个 `.yao` 文件。

```toml
# yao.mod
[module]
name = "my_project"
version = "1.0.0"
edition = "2026"

[dependencies]
"yao.std.http" = "1.0"
```

## 11.2 模块

每个 `.yao` 文件是一个模块。

```
my_project/
├── yao.mod
├── src/
│   ├── lib.yao
│   ├── client.yao
│   └── server.yao
```

## 11.3 use 导入

```yao
import yao.std.http;                              // 整个模块
import yao.std.http::{Client, Request};           // 选择性
import yao.std.http::Client as HttpClient;        // 重命名

use crate::utils;                                  // 当前包根
use super::types;                                  // 父模块
use self::helpers;                                 // 当前模块
```

## 11.4 可见性

```yao
pub fn public_fn() { ... }
fn private_fn() { ... }
pub(crate) fn crate_fn() { ... }                  // 当前 crate
pub(super) fn super_fn() { ... }                  // 父模块
pub(in path::to) fn path_fn() { ... }
```
