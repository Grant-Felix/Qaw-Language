## 15.1 声明宏

```yao
macro vec!() { /* 标准库实现 */ }

macro vec!($($x:expr),* $(,)?) => {
    {
        let mut temp = [];
        $( temp.push($x); )*
        temp
    }
};

let v = vec![1, 2, 3];
```

## 15.2 过程宏

```yao
// 在独立 proc-macro crate 中
@proc_macro
fn sql(input: TokenStream) -> TokenStream {
    // ...
}

// 在用户代码中
let x = sql!(SELECT * FROM users);
```

## 15.3 派生宏

```yao
@proc_macro_derive(Display)
fn derive_display(input: TokenStream) -> TokenStream {
    // ...
}

#[derive(Display)]
struct Point {
    x: f64,
    y: f64,
}
```

## 15.4 `quote!`

```yao
@proc_macro
fn make_struct(input: TokenStream) -> TokenStream {
    let name = parse(input).ident;
    quote! {
        struct #name {
            value: int,
        }
    }
}
```

## 15.5 内建宏

| 宏 | 用途 |
|:---|:---|
| `vec![]` | 创建 Vec |
| `format!()` | 格式化 |
| `panic!()` | panic |
| `unreachable!()` | 不可达 |
| `todo!()` | 占位 |
| `include!()` | 包含文件 |
| `include_str!()` | 包含为字符串 |
| `include_bytes!()` | 包含为字节 |
| `concat!()` | 字符串拼接 |
| `env!()` | 环境变量 |
| `cfg!()` | 编译期 cfg |
