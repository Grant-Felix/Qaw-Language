# 三大内存模式

[← 返回目录](../1-妖文编程语言从入门到精通.md)

## 8.1 @safe 模式（默认）

零运行时开销，编译期所有权检查。90% 的代码用这种。

```yao
@safe
type LinearNode {
    var val int;
    var next *LinearNode;    // 单一所有者
}
```

## 8.2 @shared 模式

自动引用计数（ARC）。允许循环引用（用 Weak 打破）。

```yao
@shared
type GraphNode {
    var val int;
    var children []GraphNode;    // 互相引用
}
```

## 8.3 @dynamic 模式

关闭编译期类型检查，使用 GC。便于与 Python 互操作。

```yao
@dynamic
func quick_parse(data any) -> any {
    return data.get("key") ?? "default";
}
```

## 8.4 模式转换

```yao
// @safe → @shared
@shared {
    let arc_x = x.arc();   // 显式转 Arc
}

// @shared → @safe
@shared {
    let owned = arc_x.unwrap_or_clone();
}

// @safe ↔ @dynamic
@safe {
    let boxed = any.box(42);   // 装箱
}

@dynamic {
    let typed = any_val.unbox::<int>();   // 拆箱
}
```

完整规则见本节与第 7 章"所有权与借用"。
