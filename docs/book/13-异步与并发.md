## 13.1 三种并发模型

```
协程 (coroutine)
├── 轻量级线程（M:N 调度）
├── go 关键字启动
└── chan 通信

异步任务 (async)
├── async/await
└── Future 状态机

线程 (thread)
└── OS 原生线程（底层）
```

## 13.2 协程

```yao
go {
    print("协程 1");
}

go compute_async(data);   // 启动协程

let handle = go task();
let result = handle.join();
```

## 13.3 通道

```yao
let c = make(chan<int>);           // 无缓冲
let cb = make(chan<string, 100>);  // 缓冲 100

c <- 42;                  // 发送
let v = <-c;              // 接收
c.close();                 // 关闭
```

## 13.4 select

```yao
select {
    v = c1.recv() => print("c1: ${v}"),
    v = c2.recv() => print("c2: ${v}"),
    _ = sleep(5.seconds) => print("timeout"),
}
```

## 13.5 async/await

```yao
async func fetch(url: string) -> Result<string, Error> {
    let resp = http.get(url).await?;
    Ok(resp.text().await?)
}

let result = fetch("https://example.com").await;
```

## 13.6 同步原语

```yao
let m = Mutex::new(0);
{
    let mut g = m.lock();
    *g += 1;
}   // g drop，锁释放

let sem = Semaphore::new(10);
let permit = sem.acquire().await;
```
