## 4.1 注释

```yao
// 单行注释

/*
多行注释
支持嵌套：/* 嵌套 */
*/

/// 文档注释（紧贴函数/结构体等）
/// 支持 Markdown 子集
```

## 4.2 标识符

支持任意 Unicode 字母（含中文）：

```yao
let 中文变量 = 1;
let café = "coffee";
let 用户_123 = "alice";
```

## 4.4 变量声明

```yao
var x int = 10;          // 显式类型
var y = 20;              // 类型推导（默认 int）
const PI = 3.14159;      // 常量（不可变）

let name = "张三";        // 不可变
let mut count = 0;       // 可变
```

## 4.5 基础类型

| 类型 | 位宽 | 范围 |
|:---|:---|:---|
| `int` / `isize` | 平台字长 | 平台相关 |
| `i8` / `i16` / `i32` / `i64` / `i128` | 8-128 | 固定 |
| `uint` / `usize` / `u8` / ... / `u128` | 平台/8-128 | 无符号 |
| `f32` / `f64` | 32 / 64 | IEEE 754 |
| `bool` | 1 | true / false |
| `string` | 变长 | UTF-8 |
| `rune` | 32 | Unicode 标量值 |
| `byte` | 8 | 0-255 |

## 4.6 运算符（节选）

完整表见附录 B。

```yao
let a = 10 + 5;      // 15
let b = a * 2;       // 30
let c = a / 3;       // 5（整数除法截断）
let d = a % 3;       // 1
let e = -a;          // -15
let f = a == b;      // false
let g = a < b;       // true
let h = !g;          // false（逻辑非）
```

## 4.7 字符串

```yao
let s = "hello, 世界";        // 普通字符串（&str）
let s2: string = s.to_string();  // String（堆分配）

// 字符串插值
let name = "张三";
print("你好, ${name}!");     // 你好, 张三!
print("结果: ${1 + 2}");     // 结果: 3

// 多行
let s3 = "第一行
第二行";

// 原始字符串
let s4 = r"不解析\转义也不${插值}";
let s5 = r#"可以包含 "双引号""#;
```
