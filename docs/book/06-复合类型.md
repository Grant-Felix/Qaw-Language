## 6.1 数组

```yao
let arr: [int; 5] = [1, 2, 3, 4, 5];
let m: [[int; 3]; 2] = [[1, 2, 3], [4, 5, 6]];
```

## 6.2 切片

```yao
let s: [int] = arr[1..3];     // 半开区间
let s: [int] = arr[1..=3];    // 闭区间
```

## 6.3 字符串切片

```yao
let s: &str = "hello";
let sub: &str = &s[0..3];     // "hel"
```

## 6.4 元组

```yao
let tup: (int, string, bool) = (42, "hi", true);
let (a, b, c) = tup;
let (_, x, _) = tup;
```

## 6.5 映射

```yao
let m: map[string]int = {"a": 1, "b": 2};
m.insert("c", 3);
let v = m.get("a").unwrap_or(0);
```

## 6.6 结构体

```yao
struct Point {
    pub x: f64,
    pub y: f64,
}

let p = Point { x: 1.0, y: 2.0 };
print(p.x);

// 字段简写
let x = 1.0;
let y = 2.0;
let p2 = Point { x, y };

// 解构
let Point { x, y } = p;
```

## 6.7 枚举

```yao
enum Shape {
    Circle(f64),
    Rectangle { width: f64, height: f64 },
    Triangle,
    Empty,
}

let s = Shape::Circle(5.0);

match s {
    Shape::Circle(r) => print("圆: ${r}"),
    Shape::Rectangle { width, height } => print("矩形: ${width}x${height}"),
    Shape::Triangle => print("三角形"),
    Shape::Empty => print("空"),
}
```

## 6.8 Option 与 Result

```yao
enum Option<T> {
    Some(T),
    None,
}

enum Result<T, E> {
    Ok(T),
    Err(E),
}

let x: Option<int> = Some(5);
let r: Result<int, string> = Ok(42);
```
