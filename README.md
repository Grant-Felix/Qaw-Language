# 妖文编程语言（Yao Language）

> **十年立项，十年走向成熟，脚踏实地。**

妖语言把人类编程史上被反复验证为"最好用"的功能，用一套统一的语法皮层标准化。开发者用一套语法就能用上 Rust 的所有权、Go 的协程、Python 的生态、C 的 FFI，不用为不同场景切换语言。

## 实现策略

- **当前阶段**：编译器 `yaoc` 用 **C / C++** 实现（参考 GCC 思路）
- **远期目标**：妖语言成熟稳定后，**逐步用妖语言自身重写** `yaoc` 各模块
- 重写完成的模块从 C 版退役，保留作为参考实现与验证基准

## 当前进度

- ✅ **v0.0** 蓝图阶段：完成完整设计文档
- ⬜ **v0.1** POC：最小可执行验证（待启动）
